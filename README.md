## セットアップ方法

```
# パッケージのインストール
pnpm install

# Next.js起動
pnpm dev

# Storybook起動
pnpm storybook

# Vitestのテスト実行
pnpm test

※その他のコマンドは、package.jsonに記載
```

## 使用技術

- TypeScript
- Next.js（App Router）
- Mantine v7
- Tailwind CSS
- Storybook
- Vitest
